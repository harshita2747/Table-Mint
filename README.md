# 🍽️ Table Mint

A backend-focused restaurant management application built with **Node.js**, **Express.js**, **MySQL**, **EJS**, and **Tailwind CSS**. The system allows users to browse the menu, view dish details, reserve tables, and save favorite dishes, while providing administrators with complete menu management through CRUD operations.

## ✨ Features

* 🍴 Browse restaurant menu
* 📖 View detailed dish information
* ❤️ Add and remove favorite dishes
* 📅 Make table reservations
* ➕ Add new menu items
* ✏️ Edit existing menu items
* 🗑️ Delete menu items
* 🗄️ MySQL database integration
* 🏗️ MVC architecture
* ⚡ Server-side rendering with EJS
* 📱 Responsive interface using Tailwind CSS
